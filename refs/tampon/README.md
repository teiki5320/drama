# `refs/tampon/` — réservoir d'images

Pool d'images générées (par toi pour ce projet ou pour d'autres) qui
**pourraient servir** mais ne sont pas encore affectées à un personnage ou
une scène. Quand on identifie un usage, on déplace vers
`assets/photos/characters/` ou `refs/scenes/` avec le bon nom.

Convention de nommage : description visuelle neutre, pas d'hypothèse de
perso. Ex. `homme_asia_30_hoodie.png` plutôt que `wenbo_jeune.png`.
