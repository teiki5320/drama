# `refs/tampon/` — réservoir d'images

Zone d'arrivée des nouvelles photos générées. À chaque livraison, je
trie :
- **Sert pour une scène / un perso identifié** → promu vers
  `assets/photos/scenes/` ou `assets/photos/characters/` avec le nom
  final attendu.
- **Variante / hors-univers / pas identifiée** → déplacé vers
  `refs/archive/` (avec un nom descriptif).

Le tampon est censé être vide ou presque entre deux livraisons.

## Workflow

1. Tu déposes tes nouveaux fichiers ici (n'importe quel nom — openart,
   IMG_xxxx, ce que ton outil te donne).
2. Tu me pings ("j'ai poussé des photos dans tampon").
3. Je lis chaque fichier, j'identifie ce que ça représente, je
   renomme et je déplace au bon endroit.
4. Je te donne un tableau récap.
