# Prompts génération d'image — personnages

Format : pour chaque perso, 1 prompt principal copy-paste, 1 prompt court
de variation, 1 prompt négatif. Tous en anglais (meilleurs résultats sur
la plupart des modèles : OpenArt, Midjourney, SDXL, FLUX). Rajoute en
suffixe ton aspect ratio (`--ar 1:1`) et style si ton outil le demande.

**Style commun** : portrait éditorial photo réaliste, lumière douce,
grain léger 35mm, fond neutre flou, pas glamour, peau réelle.

## Réglages conseillés (pour ton outil de génération)

- **Aspect ratio** : 1:1 carré (Insta, fiche profil).
- **Résolution** : ≥ 1024×1024.
- **Style** / preset : "photographic", "realistic", "editorial portrait",
  jamais "anime", "cinematic 3D", "fantasy".
- **Steps** : 30-50 (SDXL/FLUX), valeur par défaut OpenArt.
- **CFG / guidance** : 5-8.
- **Seed** : note la seed de ce qui te plaît, on pourra itérer dessus.
